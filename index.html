Module Overview

Every module is designed for real-world gym operations.

1 Membership Management

  • Member registration with auto-generated unique codes
  • Subscription plans: Daily, Weekly, Monthly, Quarterly, Half-Yearly, Annual
  • Subscription renewals with payment tracking
  • Weight loss management with BMI calculation and progress charts
  • Emergency contacts, health notes, fitness goals

2 Attendance with ZKTeco Biometric Devices

  • Real-time attendance tracking via ZKTeco fingerprint/face devices
  • Full CRUD on device users: Create, modify, enable/disable, delete users
  • Device management: Test connection, sync users, restart, configure
  • Manual check-in/check-out support with type tracking
  • Auto-checkout and late arrival detection

3 Point of Sale

  • Product sales with inventory management and low stock alerts
  • Subscription sales and renewals at the counter
  • Shopping cart with tax calculation and discounts
  • Multiple payment methods: Cash, Card, M-Pesa, Mobile Money, Bank Transfer
  • Invoice generation with printable receipts

4 Communication

  • Bulk SMS messaging with personalized variables
  • Bulk email campaigns with HTML templates
  • WhatsApp Business API integration ready
  • Message templates with variables ({first_name}, {gym_name})
  • Chat functionality for member-staff communication

Installation Guide

Get your gym management system running in minutes.

1System Requirements

  • PHP 7.4+ (PHP 8.x recommended)
  • MySQL 5.7+ or MariaDB 10.2+
  • Apache with mod_rewrite enabled
  • ZKTeco device (optional, for biometric attendance)

2Database Setup

mysql -u root -p
CREATE DATABASE gym_management CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE gym_management;
SOURCE database.sql;

3Configure Database

Edit config/database.php with your credentials:

define('DB_HOST', 'localhost');
define('DB_NAME', 'gym_management');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');

4Login

Access the system and login with default credentials:

Username: superadmin

Password: password

Important: Change default password after first login!
